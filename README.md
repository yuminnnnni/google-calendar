# 📅 Google Calendar Clone

## 🔧 프론트엔드 라이브러리

- **React + TypeScript**
- **Redux Toolkit** (전역 상태 관리)
- **React Day Picker** (날짜 선택 UI 컴포넌트)
- **lucide-react** (아이콘 사용)

## ⚙️ 유틸리티

- **date-fns** (날짜 계산 및 포맷팅)
- **uuid** (고유 ID 생성)

## 🛠️ 개발 도구

- **Vite**
- **TypeScript**

## 📦 설치 및 실행

> **⚠️ 꼭 아래 명령어를 먼저 실행해야 정상적으로 프로젝트가 동작합니다!**

```bash
npm install
npm run dev
```
